# 🎵 Flask Music Catalogue Management System

A web-based **Music Catalogue Management System** built using **Python** and **Flask** to provide an efficient way of organizing and managing music collections. The application enables users to add, update, search, view, and delete music records through a clean and interactive web interface while demonstrating the practical use of Flask, relational databases, and full-stack web development concepts.

This project was developed as part of a collaborative academic software engineering project and showcases my ability to build dynamic web applications using Python and modern backend development practices.

This is an updated version of the Java Based Model
---

## 📖 Project Overview

The Flask Music Catalogue Management System is designed to simplify the management of digital music libraries by providing a centralized platform for maintaining music records. Users can easily perform essential catalogue operations, making it easier to organize and retrieve music information whenever needed.

The project combines backend development with database management and responsive web design to deliver a functional and user-friendly application.

---

## ✨ Features

* 🎼 Add new music records
* 🔍 Search songs by title, artist, album, or genre
* ✏️ Update existing music information
* 🗑️ Delete music records
* 📋 View the complete music catalogue
* 💾 Store and manage data using a relational database
* 🌐 Interactive web interface built with Flask
* ⚡ Fast and efficient catalogue management

## 🚀 Getting Started

### Prerequisites

Before running the application, ensure you have:

* Python 3.8 or later
* Flask
* Required project dependencies

### Installation

Clone the repository:

```bash
git clone https://github.com/NancyMerciline/Flask_Music_Catalogue_Management_System.git
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the Flask application:

```bash
flask run
```

Open your browser and navigate to:

```text
http://127.0.0.1:5000/
```
